# Agent Skill Infrastructure

Agent Skill development infrastructure: quality check, behavior test runner, and version awareness.

## Modules

- **quality-check** — L1/L2 quality scoring using helloandy 8-dimension framework
- **test-runner** — Behavior test runner with evals.json format and snapshot testing
- **version-aware** — Git-backed version management with change summaries and rollback

## Quick Start

```bash
uv sync
uv run pytest
```
